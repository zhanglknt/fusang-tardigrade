# Fusang: Tardigrade Edition

**Alignment-free phylogenetic inference via spaced k-mer divide-and-conquer.**

Resilient to indels, scalable to 10,000+ taxa.

## Official Names

- **Full name**: *Fusang: Tardigrade Edition*
- **Short name**: `fusang-tardigrade`

## Features

- Spaced k-mer frequency vectors (k=5, gap2 default)
- Cosine distance + Neighbor-Joining (NJ) tree building
- Divide-and-conquer architecture for large-scale inference
- Robust to insertion/deletion (indel) evolution

## Quick Start

```bash
# Install
pip install -e .

# Run
python fusang_v2.py --help
```
